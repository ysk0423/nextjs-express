# Next.js ✖️ Express Sandbox

## Getting Started

```bash
docker compose up
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Library

```zsh
yarn add express
yarn add -D nodemon ts-node @types/express @types/node
```
